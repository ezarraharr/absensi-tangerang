# 📋 Sistem Absensi Karyawan - Tangerang

Aplikasi Absensi sederhana berbasis web yang bisa diinstall sebagai aplikasi di HP.

![Absensi](https://img.shields.io/badge/Status-Live-brightgreen)
![GitHub Pages](https://img.shields.io/badge/Hosted%20on-GitHub%20Pages-blue)

## ✨ Fitur

- ✅ Deteksi otomatis terlambat (08.00 tidak terlambat)
- ✅ Pilih nama karyawan dengan tombol (Farhan & Adit)
- ✅ Tampilan waktu real-time (jam, hari, tanggal, bulan, tahun)
- ✅ Riwayat absensi yang tersimpan di browser
- ✅ Bisa diinstall sebagai aplikasi di HP (PWA)
- ✅ Tampilan modern & responsif

## 🚀 Link Aplikasi

👉 **Akses Aplikasi:** [https://ezarraharr.github.io/absensi-tangerang](https://ezarraharr.github.io/absensi-tangerang)

## 📱 Cara Install di HP

1. Buka link di atas menggunakan **Google Chrome**
2. Klik ⋮ (tiga titik) di pojok kanan atas
3. Pilih **"Install app"** atau **"Tambahkan ke layar utama"**
4. Selesai! Aplikasi sudah ada di home screen HP kamu

## 🕒 Aturan Absensi

- **Sebelum pukul 08.00** → ✅ Tidak Terlambat
- **Setelah pukul 08.01** → ⚠️ Terlambat
